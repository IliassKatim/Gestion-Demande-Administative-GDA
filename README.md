# Gestion-Demande-Administative-GDA
Le projet consiste en la conception et le développement d'une plateforme web dédiée à la simplification des démarches administratives au Maroc. Cette plateforme permet aux citoyens d'obtenir des documents officiels tels que des extraits de naissance, des certificats de résidence et des actes de mariage sans avoir à se déplacer physiquement. L'objectif principal est de numériser les processus administratifs pour gagner en efficacité, réduire les délais de traitement et améliorer l'expérience utilisateur. Le système inclut une interface utilisateur intuitive, un espace administrateur pour la gestion des demandes, et une base de données pour stocker les informations des utilisateurs et des demandes.
